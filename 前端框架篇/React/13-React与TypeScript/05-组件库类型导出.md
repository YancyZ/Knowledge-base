# 组件库类型导出

> 做 **UI 组件库** 或 **内部 design system** 时，类型导出与 `package.json exports` 同样重要——消费者靠 `.d.ts` 获得智能提示。

---

## 一、包结构

```
my-ui/
├── src/
│   ├── Button/
│   │   ├── Button.tsx
│   │   └── index.ts
│   └── index.ts
├── dist/
│   ├── index.js
│   └── index.d.ts
└── package.json
```

```json
{
  "name": "@acme/ui",
  "main": "./dist/index.js",
  "types": "./dist/index.d.ts",
  "exports": {
    ".": {
      "types": "./dist/index.d.ts",
      "import": "./dist/index.js"
    }
  }
}
```

---

## 二、 barrel export

```tsx
// src/index.ts
export { Button } from './Button';
export type { ButtonProps } from './Button';

export { ThemeProvider, useTheme } from './theme';
export type { Theme } from './theme';
```

| 实践 | 说明 |
|------|------|
| `export type` | 仅类型，运行时无开销 |
| 避免 `export *` 过度 | 控制公共 API 面 |

## 三、Props 导出命名

```tsx
// Button.tsx
export interface ButtonProps extends React.ComponentPropsWithoutRef<'button'> {
  variant?: 'primary' | 'secondary';
  size?: 'sm' | 'md' | 'lg';
}

export function Button(props: ButtonProps) { ... }
```

消费者：

```tsx
import { Button, type ButtonProps } from '@acme/ui';

const props: ButtonProps = { variant: 'primary', children: 'OK' };
```

## 四、Variant 类型共享

```tsx
export const buttonVariants = ['primary', 'secondary', 'ghost'] as const;
export type ButtonVariant = (typeof buttonVariants)[number];
```

与 CVA（class-variance-authority）配合：

```tsx
import { cva, type VariantProps } from 'class-variance-authority';

const button = cva('btn', {
  variants: {
    variant: { primary: 'btn-primary', secondary: 'btn-secondary' },
    size: { sm: 'btn-sm', md: 'btn-md' },
  },
  defaultVariants: { variant: 'primary', size: 'md' },
});

export type ButtonProps = React.ComponentPropsWithoutRef<'button'> &
  VariantProps<typeof button>;
```

## 五、Polymorphic `as` 类型

```tsx
type AsProp<C extends React.ElementType> = { as?: C };

type PropsToOmit<C extends React.ElementType, P> = keyof (AsProp<C> & P);

type PolymorphicComponentProp<
  C extends React.ElementType,
  Props = {},
> = React.PropsWithChildren<Props & AsProp<C>> &
  Omit<React.ComponentPropsWithoutRef<C>, PropsToOmit<C, Props>>;

export function Box<C extends React.ElementType = 'div'>({
  as,
  ...props
}: PolymorphicComponentProp<C, { className?: string }>) {
  const Component = as ?? 'div';
  return <Component {...props} />;
}
```

复杂但 shadcn / Radix 同类组件常用。

## 六、构建与 tsc

| 工具 | 说明 |

## 七、peerDependencies

```json
{
  "peerDependencies": {
    "react": "^18.0.0 || ^19.0.0",
    "react-dom": "^18.0.0 || ^19.0.0"
  },
  "devDependencies": {
    "@types/react": "^18.3.0",
    "react": "^18.3.0"
  }
}
```

库不打包 React，类型对齐 consumer 的 `@types/react`。
