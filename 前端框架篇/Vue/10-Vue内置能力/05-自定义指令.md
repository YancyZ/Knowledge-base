# 自定义指令

**自定义指令**对 DOM 施加底层行为（focus、permission、click-outside），钩子 **mounted/updated/unmounted**，unmounted 务必清理副作用；复杂逻辑优先 composable。

---

## 指令 vs composable

| 指令 | composable |
|------|------------|
| 单一 DOM 元素行为 | 组件内多 state 逻辑 |
| 第三方 DOM 集成 | 可测试纯函数 |
| v-xxx 声明式 | useXxx 显式调用 |

```vue
<input v-focus />
<button v-permission="'admin:delete'">删除</button>
```

---

## 注册方式

**全局：**

```js
// main.js
import { createApp } from 'vue'

const app = createApp(App)

app.directive('focus', {
  mounted(el) {
    el.focus()
  }
})

app.mount('#app')
```

**局部（script setup）：**

```vue
<script setup>
const vFocus = {
  mounted: (el) => el.focus()
}
</script>

<template>
  <input v-focus />
</template>
```

---

## 钩子与参数

```js
app.directive('demo', {
  mounted(el, binding, vnode) {
    // binding.value 指令值
    // binding.arg 指令参数 v-demo:foo
    // binding.modifiers 修饰符对象
  },
  updated(el, binding) {
    // 所在组件更新后
  },
  unmounted(el) {
    // 清理监听器
  }
})
```

Vue 3 移除 **bind / insert / componentUpdated** 等 Vue 2 名，统一为 **mounted / updated / unmounted**。

---

## 示例：v-click-outside

```js
const clickOutside = {
  mounted(el, binding) {
    el._clickOutside = (e) => {
      if (!el.contains(e.target)) {
        binding.value(e)
      }
    }
    document.addEventListener('click', el._clickOutside)
  },
  unmounted(el) {
    document.removeEventListener('click', el._clickOutside)
  }
}
```

```vue
<div v-click-outside="closeMenu">...</div>
```

**unmounted 必须清理** document 级监听。

---

## 示例：v-permission

```js
function permission(el, binding) {
  const allowed = checkPermission(binding.value)
  el.style.display = allowed ? '' : 'none'
  // 或 el.disabled / el.remove()
}

export const vPermission = {
  mounted: permission,
  updated: permission
}
```

权限变更时 **updated** 重新计算。

---

## 示例：v-lazy（IntersectionObserver）

```js
const lazy = {
  mounted(el, binding) {
    const src = binding.value
    const io = new IntersectionObserver(([entry]) => {
      if (entry.isIntersecting) {
        el.src = src
        io.disconnect()
      }
    })
    io.observe(el)
    el._io = io
  },
  unmounted(el) {
    el._io?.disconnect()
  }
}
```

---

## 与组件的区别

指令**不能**用于组件根以外的自定义组件内部根（组件有多个根时无单一 el）；仅适用于**元素**或组件根元素。

## TypeScript augmentation

```ts
// directives.d.ts

  interface ComponentCustomProperties {
    vPermission: Directive<string>
  }
}
```

## Vue 2 迁移

| Vue 2 | Vue 3 |
|-------|-------|
| bind → inserted | mounted |
| unbind | unmounted |
| 函数式 directive | 仍支持简写函数（仅 mounted/unmounted） |

---

## 小结

**适用场景**：DOM 级横切，focus、permission、click-outside、lazy load 等单一元素行为。

**钩子**：mounted / updated / unmounted；Vue 2 的 bind/unbind 等已更名。

**binding**：value（指令值）、arg（参数）、modifiers（修饰符）。

**清理**：document/window 级监听、IntersectionObserver 等须在 **unmounted** 移除，常见泄漏源。

**示例模式**：click-outside 挂 document click；permission 在 updated 重算；lazy 用 IO 并在 unmounted disconnect。

**vs composable**：多 state 逻辑、可测试纯函数优先 composable；指令保持薄、声明式。

**组件限制**：指令作用于元素或组件根；多根组件无单一 el。

**注册**：app.directive 全局或 script setup 局部 vXxx 变量。

**Vue 2 迁移**：bind→mounted，unbind→unmounted。
