# 动态规划入门

**动态规划（DP）** 存重叠子问题的最优解，避免指数重复递归。背包、LIS、编辑距离，状态可枚举、转移清晰时，DP 把暴力压到多项式时间。

---

## 识别 DP

1. **最优子结构**
2. **重叠子问题**
3. **无后效性**

```mermaid
flowchart LR

  Rec --> Memo[记忆化]
  Memo --> Tab[递推填表]
```

| 信号 | 说明 |
|------|------|
| 问「最多/最少/方案数」 | 常是 DP |
| 递归树重复节点 | 记忆化 |
| 选或不选 | 背包类 |

## 两种实现

| | top-down | bottom-up |
|---|----------|-----------|
| 写法 | 递归+Map | for 循环 |
| 栈 | 有 | 无 |
| 空间 | 递归栈 | 可滚动优化 |

```javascript
function fibTab(n) {
  if (n <= 1) return n;
  let a = 0, b = 1;
  for (let i = 2; i <= n; i++) [a, b] = [b, a + b];
  return b;
}
```

## 一维 DP

`dp[i]` 含义明确 → 转移 → 初值 → 顺序。

```javascript
function rob(nums) {
  let prev2 = 0, prev1 = 0;
  for (const x of nums) {
    const cur = Math.max(prev1, prev2 + x);
    prev2 = prev1; prev1 = cur;
  }
  return prev1;
}
```

爬楼梯、打家劫舍、最大子数组（Kadane 可视为 dp）。

## 二维 DP

**LCS** — diff 算法基础之一。

```javascript
function lcs(a, b) {
  const m = a.length, n = b.length;
  const dp = Array.from({ length: m + 1 }, () => Array(n + 1).fill(0));
  for (let i = 1; i <= m; i++)
    for (let j = 1; j <= n; j++)
      dp[i][j] = a[i-1] === b[j-1]
        ? dp[i-1][j-1] + 1
        : Math.max(dp[i-1][j], dp[i][j-1]);
  return dp[m][n];
}
```

0-1 背包、编辑距离、路径数。

## 0-1 背包要点

```javascript
// dp[w] = 容量 w 的最大价值
for (const item of items) {
  for (let w = cap; w >= item.weight; w--)
    dp[w] = Math.max(dp[w], dp[w - item.weight] + item.val);
}
// 内层倒序 — 每个物品只用一次
```

完全背包内层正序；0-1 与完全差别在循环方向。

## 空间优化

只依赖上一行 → 滚动数组一行。LCS 可压成 O(min(m,n))。

## 一维 DP 模板

```javascript
// 爬楼梯：dp[i] = dp[i-1] + dp[i-2]
function climb(n) {
  let a = 1, b = 1;
  for (let i = 2; i <= n; i++) [a, b] = [b, a + b];
  return b;
}
```

定义状态、转移、初值、遍历顺序四步；空间可滚动数组优化。

## 例题：0-1 背包

`dp[j]` = 容量 j 时最大价值；物品 i 只能选一次，**倒序**遍历容量：

```javascript
function knapsack01(weights, values, cap) {
  const dp = Array(cap + 1).fill(0);
  for (let i = 0; i < weights.length; i++) {
    for (let j = cap; j >= weights[i]; j--) {
      dp[j] = Math.max(dp[j], dp[j - weights[i]] + values[i]);
    }
  }
  return dp[cap];
}
```

| 背包类型 | 内层 j 方向 |
|----------|-------------|
| 0-1 | 倒序 |
| 完全（无限件） | 正序 |

前端资源加载「总带宽 cap 下选文件包最大化体验分」可抽象为背包，实际工程多用启发式，DP 用于理解取舍。

## LCS 最长公共子序列

```javascript
function lcs(a, b) {
  const dp = Array.from({ length: a.length + 1 }, () => Array(b.length + 1).fill(0));
  for (let i = 1; i <= a.length; i++)
    for (let j = 1; j <= b.length; j++)
      dp[i][j] = a[i - 1] === b[j - 1]
        ? dp[i - 1][j - 1] + 1
        : Math.max(dp[i - 1][j], dp[i][j - 1]);
  return dp[a.length][b.length];
}
```

Git diff、文本相似度、版本合并都涉及 LCS 或其变体。

## 小结

DP = 最优子结构 + 重叠 + 转移；先定义 `dp` 含义再写循环。

**易混点**：子序列 vs 子串；0-1 与完全背包循环方向相反；记忆化不等于 DP 表格。
